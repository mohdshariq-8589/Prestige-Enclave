
   ```env
   PORT=5000
   ```

   ```env
   VITE_FIREBASE_API_KEY=your_api_key
   ```

4. **Run the App**

     ```bash
     cd backend
     npm run dev
     ```
   - Start Frontend:
     ```bash
     cd ../client
     npm run dev
     ```
